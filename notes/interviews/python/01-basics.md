# Python Overview And Syntax

#### Python 是什么？

- 高级的、面向对象的、跨平台的、解释型的编程语言
- 内置高级数据结构，结合动态类型和动态绑定，广泛用于数据科学和人工智能领域

#### 直接调用 `[]` 优于 `list()`

- `[]` 快于 `list()`
- `[]` 步骤少，`list()` 会调用 `dis()`，字节码比 `[]` 多一行

#### Python 有哪些应用？

- 数据科学数据分析
- NLP
- 图像处理
- 网络和互联网开发
- 游戏开发
- 应用程序开发
- GUI 的桌面应用程序开发

> 流行的 Python Web 框架有 Django、Flask、FastAPI
> Python 的标准库支持电子邮件处理、FTP、IMAP 和其他 Internet 协议
> Python 的 `scipy` 和 `numpy` 有助于科学和计算应用程序的开发
> Python 的 `tkinter` 库支持创建基于桌面的 GUI 应用程序

#### Python 有什么优势？

- 高级的、面向对象的、跨平台的、解释型的编程语言
- 解释型语言，不需要事先编译代码，直接执行指令
- 免费和开源
- 社区活跃
- 具有内置的数据结构
- 可读性
- 语法简单，快速开发

#### PEP 是什么？

- Python 代码的指南的文档，也是一组规则，指定如何格式化 Python 代码以获得最大的可读性

#### Python 中的字面量是什么？

- 字面量可以定义为以变量或常量给出的数据
- 包括字符串文字、数字文字、布尔文字、特殊文字

#### Python 中的函数是什么？

- 函数是程序的一部分或代码块，只需编写一次，就可以在程序需要时随时执行
- 函数是具有有效名称、参数列表、返回值和主体的自包含语句块
- 函数使编程更具功能性和模块化，以执行模块化任务
- 共有 `3` 种类型的功能，包括内置函数、自定义函数、匿名函数

#### Python 中的 `zip()` 函数是什么？

- `zip()` 函数返回一个 `Zip` 对象，该对象映射多个容器的相似索引
- 它从两个或多个迭代器返回一个迭代器
- 根据相似索引聚合参数的元素

#### Python 中的 `swapcase()` 函数是什么？

- 返回一个字符串对象，字符串内容为传参字符串的大小写转换

#### Python 中删除字符串前后空格？

- 使用 `strip()` 函数，返回一个字符串对象，字符串内容将传参字符串前后空格删除
- 使用 `lstrip()` 函数，删除前导空格
- 使用 `rstrip()` 函数，删除尾随空格

#### 举个 `shuffle()` 方法的例子？

- 此方法随机播放给定的字符串或数组，它随机化数组中的项目
- 此方法存在于随机模块中
- 每次函数调用并产生不同的输出时，它都会打乱元素

#### Python 中的 `break` 语句有什么用？

- 用于终止当前循环的执行，总是中断当前执行并将控制转移到当前块之外
- 如果块在循环中，则从循环中退出，如果中断在嵌套循环中，则从最里面的循环中退出

#### Python 中的元组是什么？

- 元组是一种内置的数据结构
- 按顺序存储值，且不可变
- 使用括号创建元组
- 可以使用索引来获取元素
- 元组支持各种方法，如 `max()`、`sum()`、`sorted()`、`len()` 等

#### Python 支持哪些不同的文件处理模式？

- `w`，写入模式，当需要更改文件和更改或添加信息时使用此模式。请记住，这将擦除现有文件以创建一个新文件。文件指针位于文件的开头
- `r`，读取模式，当文件中的信息仅用于读取且不以任何方式更改时使用此模式。文件指针位于文件的开头
- `a`，附加模式，此模式自动在文件末尾添加信息。文件指针放置在文件的末尾
- `r+`，读/写模式，当你要对文件进行更改并从中读取信息时使用。文件指针位于文件的开头
- `w+`, 读/写模式
- `a+`，追加和读取模式，打开一个文件，允许数据被添加到文件的末尾，也让你的程序读取信息。文件指针放置在文件的末尾

> `r+`，可读可写，若文件不存在，报错
> `w+`, 可读可写，若文件不存在，创建

#### Python 中的运算符？

- 运算符是用于某些值并产生输出结果的特定符号
- 运算符处理操作数，操作数是包含一些值的数字文字或变量
- 运算符可以是一元、二元或三元。需要单个操作数的运算符称为一元运算符，需要两个操作数的称为二元运算符，需要三个操作数的运算符称为三元运算符

#### 如何在 Python 中创建 `unicode` 字符串？

- 在 Python 3 中，旧的 `unicode` 类型已经替换为 `str` 类型，字符串默认被视为 `unicode`
- 可以使用 `art.title.encode("utf-8")` 函数在 `unicode` 中创建一个字符串

#### Python 是解释型语言吗？

- Python 是一种解释型语言
- Python 语言程序直接从源代码运行，它将源代码转换为中间语言代码，然后再将其翻译成必须执行的机器语言。与 Java 或 C 不同，Python 在执行前不需要编译

#### Python 作用域？

- 局部作用域
	- 局部作用域通常指的是在函数内部定义的变量
	- 局部变量只能在定义它们的函数内部被访问和修改
	- 当函数执行完成后，局部变量会被销毁
- 嵌套作用域
	- 如果一个函数定义在另一个函数内部，那么这个内部函数可以访问外部函数定义的变量，这就是嵌套作用域
	- 嵌套作用域不是全局的，也不是局部的，它是介于两者之间的一种作用域
	- 修改嵌套作用域（enclosing 作用域，外层非全局作用域）中的变量则需要 `nonlocal` 关键字
- 全局作用域
	- 定义在函数外的拥有全局作用域
	- 全局变量可以在整个模块内被访问和修改
	- 如果在函数内部试图修改一个全局变量，需要使用 `global` 关键字来声明
- 内置作用域
	- Python 解释器自带的一些名字（如 `print()`, `len()` 等）位于内置作用域
	- 内置作用域中的名字在所有其他作用域中都是可见的
	- 内置作用域通常不会被用户修改

> 局部变量只能在其被声明的函数内部访问，而全局变量可以在整个程序范围内访问
> 调用函数时，所有在函数内声明的变量名称都将被加入到作用域中

#### Python 中的命名空间？

- 命名空间被定义为一个简单的系统来控制程序中的名称
- 它确保名称是唯一的并且不会导致任何冲突
- Python 以字典的形式实现命名空间，并维护名称到对象的映射，其中名称作为键，对象作为值

#### Python 中的切片是什么？

- 切片是一种用于从列表、元组和字符串等序列类型中选择一系列元素的机制
- 它需要一个冒号来分隔字段的开始和结束索引，还需要一个冒号来指定步长

#### Python 中的字典是什么？

- 字典是一种内置数据类型，它定义了键和值之间的一对一关系
- 将元素存储在键值对中
- 键是唯一的，而值可以是重复的
- 通过键访问元素

#### Python 中的 `pass` 有什么作用 ？

- 是没有操作的语句
- 复合语句的占位符

#### Python 文档字符串是什么？

- Python 文档字符串是一个字符串文字，它作为模块、函数、类或方法定义中的第一条语句出现。它提供了一种方便的方式来关联文档
- 在顶部的简单赋值之后立即出现的字符串文字称为“属性文档字符串”
- 在另一个文档字符串之后立即出现的字符串文字称为“附加文档字符串”
- Python 使用三引号来创建文档字符串，即使字符串适合一行
- Docstring 短语以句点结尾，可以是多行，它可能由空格和其他特殊字符组成

```python
class Person:
    """这是一个表示人的类。

    Attributes:
        name (str): 人的名字。
        age (int): 人的年龄。
    """

    def __init__(self, name, age):
        """初始化 Person 类的实例。

        Args:
            name (str): 人的名字。
            age (int): 人的年龄。
        """
        self.name = name
        """人的名字"""

        self.age = age
        """人的年龄"""
```

#### Python 中的负索引是什么，为什么要使用它们？

- Python 中的序列是有索引的，它由正数和负数组成。正数使用 `0` 作为第一个索引，`1` 作为第二个索引，过程就这样继续下去。负数的索引从 `-1` 开始，表示序列中的最后一个索引，`-2` 作为倒数第二个索引，序列像正数一样向前推进
- 负索引用于从字符串中删除任何换行符，并允许字符串除了作为 `S[:-1]` 给出的最后一个字符
- 负索引也用于显示索引以正确顺序表示字符串

#### Python 如何进行编译时和运行时代码检查？

- 在 Python 中，一些编码是在编译时完成的，但大多数检查 (如类型、名称等) 都推迟到代码执行时进行
- 如果 Python 代码引用了不存在的用户定义函数，则代码将编译成功

#### 打开文本文件并显示其内容的最短方法是什么？

- 使用 `with open`

```python
with open("path/to/file.txt", "r", encoding="utf-8") as file:
    file_data = file.read()

print(file_data)
```

#### Python 中 `enumerate()` 函数的用途是什么？

- 返回一个迭代器，用于遍历序列并同时检索索引位置及其对应值

#### Python 中的类型转换是什么？

- 类型转换是指将一种数据类型转换为另一种数据类型
- `int ()`，将任何数据类型转换为整数类型
- `float ()`，将任何数据类型转换为浮点类型
- `ord ()`，将字符转换为整数
- `hex ()`，将整数转换为十六进制
- `oct ()`，将整数转换为八进制
- `tuple ()`，此函数用于转换为元组
- `set ()`，此函数返回转换为集合类型
- `list()`，此函数用于将任何数据类型转换为列表类型
- `dict ()`，此函数用于将顺序的键值对元组转换为字典
- `str ()`，用于将整数转换为字符串
- `complex (real, imag)`，此函数将实数转换为复数
