# Python Notes

Python 教程、练习、资源、可运行示例和个人知识库。

## 目录

- `src/python_notes/tutorials/`: 基础语法、数据分析、工具、并发、Web、算法和机器学习教程。
- `src/python_notes/exercises/`: 配套练习、测评和项目。
- `src/python_notes/examples/`: 基础脚本和任务队列示例。
- `src/python_notes/resources/`: notebook 图片和数据集。
- `notes/`: 系统配置、数据库、面试笔记和补齐教程。
- `tests/`: 包结构、notebook 语法和示例行为检查。

## 知识库入口

- `notes/learning-roadmap.md`: 从原待办清单整理出的学习路线。
- `notes/tutorials/`: pytest、Web、爬虫、网络服务、Pydantic、UDS、量化模型等补齐教程。
- `notes/interviews/`: Python、后端、计算机基础和软件工程面试笔记。
