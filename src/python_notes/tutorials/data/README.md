# 数据分析教程

这个目录按工具分组，文件名尽量使用短词；必须表达两个并列概念时使用下划线。

## NumPy

- [[numpy.ipynb|numpy]]: 入门、ndarray 和向量化。
- [[creation.ipynb|creation]]: 数组创建。
- [[transform.ipynb|transform]]: 形状、轴和类型变换。
- [[indexing_broadcasting.ipynb|indexing_broadcasting]]: 索引、切片、布尔筛选和广播。
- [[aggregation.ipynb|aggregation]]: 聚合统计和按轴归约。
- [[combine.ipynb|combine]]: 合并、拆分、插入和删除。
- [[practice.ipynb|practice]]: NumPy 综合练习。
- [[meshgrid.ipynb|meshgrid]]: 网格坐标和索引网格。
- [[where.ipynb|where]]: 条件选择、极值位置和多维索引。
- [[sort.ipynb|sort]]: 排序、间接排序和多键排序。
- [[functions.ipynb|functions]]: `vectorize`、`apply_along_axis`、`piecewise`、`digitize` 和 `clip`。

## Pandas

- [[pandas.ipynb|pandas]]: 入门、Series 和 DataFrame 概览。
- [[series.ipynb|series]]: Series 创建和索引对齐运算。
- [[dataframe.ipynb|dataframe]]: DataFrame 创建、选择、筛选和索引。
- [[multiindex.ipynb|multiindex]]: 多级索引。
- [[merge.ipynb|merge]]: `concat`、`merge` 和 `join`。
- [[cleaning.ipynb|cleaning]]: CSV 读取、类型转换、缺失值和重复值处理。
- [[plotting.ipynb|plotting]]: Pandas 内置绘图。

## Matplotlib

- [[lines.ipynb|lines]]: 折线图。
- [[axes.ipynb|axes]]: 坐标轴、刻度和标注。
- [[subplots.ipynb|subplots]]: 子图布局。
- [[scatter.ipynb|scatter]]: 散点图。
- [[bars.ipynb|bars]]: 柱形图。
- [[histograms.ipynb|histograms]]: 直方图。
- [[pie.ipynb|pie]]: 饼图。
- [[errors.ipynb|errors]]: 误差条和误差带。
- [[contours.ipynb|contours]]: 轮廓图和图像显示。
- [[3d.ipynb|3d]]: 三维图形。

配套图片和数据集放在 [[../../resources/README|resources]]，notebook 中的相对路径已按该目录组织。
