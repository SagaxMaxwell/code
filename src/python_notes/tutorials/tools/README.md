# Python 工具模块

- [[apscheduler.ipynb|apscheduler]]: APScheduler 定时任务配置、触发器、任务管理和生命周期。
- [[math.ipynb|math]]: `math` 实数数学工具。
- [[cmath.ipynb|cmath]]: `cmath` 复数数学工具。
- [[decimal.ipynb|decimal]]: `decimal.Decimal` 十进制定点计算。
- [[fractions.ipynb|fractions]]: `fractions.Fraction` 有理数计算。
- [[calendar.ipynb|calendar]]: `calendar` 日历生成和工作日统计。
- [[numeric.ipynb|numeric]]: `math`、`cmath`、`decimal` 和 `fractions` 选型对比。
- [[pydantic.ipynb|pydantic]]: Pydantic 数据校验工具教程，包含订单模型、批量校验、JSON 序列化和错误整理。
