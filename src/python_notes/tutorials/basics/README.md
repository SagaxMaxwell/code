# Python 基础教程

这个目录按基础语法领域拆分，文件名尽量使用短词。

## 数据与表达式

- [[types.ipynb|types]]: 变量、对象引用和基本类型。
- [[operators.ipynb|operators]]: 运算符。
- [[strings.ipynb|strings]]: 字符串。
- [[bases.ipynb|bases]]: 进制转换。
- [[lists.ipynb|lists]]: 列表。
- [[tuples.ipynb|tuples]]: 元组。
- [[dicts.ipynb|dicts]]: 字典。
- [[sets.ipynb|sets]]: 集合。

## 函数与迭代

- [[functions.ipynb|functions]]: 函数定义。
- [[args.ipynb|args]]: 函数参数。
- [[returns.ipynb|returns]]: 返回值。
- [[scope.ipynb|scope]]: 作用域。
- [[lambda.ipynb|lambda]]: `lambda` 和 `map`。
- [[builtins.ipynb|builtins]]: 常用内置函数。
- [[collections.ipynb|collections]]: `collections` 标准库。
- [[itertools.ipynb|itertools]]: `itertools` 标准库。
- [[unpack.ipynb|unpack]]: 打包与解包。
- [[iterators.ipynb|iterators]]: 生成器与迭代器。

## 文件与对象

- [[files.ipynb|files]]: 文件读写。
- [[dirs.ipynb|dirs]]: 目录操作。
- [[oop.ipynb|oop]]: 面向对象概览。
- [[classes.ipynb|classes]]: 类定义。
- [[init.ipynb|init]]: 构造函数和资源管理。
- [[methods.ipynb|methods]]: 属性与方法。
- [[special.ipynb|special]]: 特殊方法和运算符重载。
- [[inheritance.ipynb|inheritance]]: 继承、组合和方法重写。
