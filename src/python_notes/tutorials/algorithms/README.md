# 算法模块

这个模块分为算法模式和算法题库。算法模式按可独立学习的题型拆分；题库文件保留来源分组，避免把大量单题拆成不可维护的小文件。

- [[window.ipynb|window]]: 滑动窗口。
- [[combinatorics.ipynb|combinatorics]]: 子集、排列和组合。
- [[search.ipynb|search]]: 变形二分查找。
- [[topk.ipynb|topk]]: Top K Elements。
- [[tree.ipynb|tree]]: 二叉树 DFS、BFS 和深度计算。
- [[graph.ipynb|graph]]: 图 BFS、DFS、Dijkstra 和拓扑排序。
- [[pointers.ipynb|pointers]]: 双指针。
- [[dp.ipynb|dp]]: 动态规划。
- [[greedy.ipynb|greedy]]: 贪心算法。
- [[backtracking.ipynb|backtracking]]: 回溯、递归和分治。
- [[sort.ipynb|sort]]: 常见排序算法。
- [[problems.ipynb|problems]]: 经典算法题库，包括 LeetCode 和 HuaWei OD 题目。
