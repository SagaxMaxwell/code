# Gradio 和 Streamlit 教程

- `gradio_streamlit.ipynb`: 面向 Python 数据应用和机器学习 Demo 的 Gradio、Streamlit 入门教程，包含组件选择、应用结构、状态、缓存和运行方式。

## 运行准备

这两个框架不是当前项目的必需运行依赖。需要实际启动示例时，先安装对应包：

```bash
uv add gradio streamlit
```

Gradio 示例通常直接运行 Python 文件或 notebook 代码单元即可启动服务。Streamlit 示例需要保存为 `.py` 文件后运行：

```bash
streamlit run app.py
```
